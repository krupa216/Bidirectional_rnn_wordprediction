**Project Title**

Next Word Prediction using Bidirectional RNN 

**Project Description**

This project focuses on building a Next Word Prediction system using a Bidirectional Recurrent Neural Network (Bi-LSTM). The goal of the system is to predict the most probable next word in a sentence based on the context of the previously typed words.

The model is trained on a text corpus where sentences are tokenized and converted into numerical sequences. These sequences are then used to train a Bidirectional Long Short-Term Memory (Bi-LSTM) network, which can learn contextual relationships in both forward and backward directions. This helps the model understand the structure of language more effectively compared to traditional RNNs.

The system uses TensorFlow and Keras for model development, along with tokenization and sequence padding techniques for text preprocessing. After training, the model is saved along with the tokenizer to enable future predictions without retraining.

**Tools Used**

Programming Language: Python

Deep Learning: TensorFlow, Keras

Model: Bidirectional LSTM

Backend: Flask

Libraries: NumPy, Flask-CORS

Model Storage: HDF5 (.h5), Pickle (.pkl)

Frontend: HTML, CSS, JavaScript
